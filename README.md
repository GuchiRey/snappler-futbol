# Snappler Futbol Challenge

Node version: 18

# 1- Install dependencies

```
npm install
```

# 2- Run project

```
npm run start
```

# 3- Run proxy server

```
git clone https://github.com/GuchiRey/server-proxy-snappler.git
```
```
cd server-proxy
```
```
npm run install
```
```
node proxy.js
```
